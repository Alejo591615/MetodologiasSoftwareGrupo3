# Comandos de Git - Alejandro Hernández

## 1. Crear una nueva rama local con mi nombre

- git checkout -b AlejandroHernandez 

Tambien se puede hacer de la siguiente forma: 

- git branch AlejandroHernandez >
git checkout AlejandroHernandez

## 2. Crear commit en la misma rama

- git add . > git commit -m "Agrega archivo de Alejandro Hernández"

## 3. Subit rama a Git

- git push -u origin AlejandroHernandez
